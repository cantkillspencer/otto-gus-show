How to customize

🎨 Change a color

Open css/colors.css. Find the variable. Change the hex value. Save. Done.

➕ Add an episode

Open episodes.html. Copy any <article class="episode-card"> block. Paste it into the grid. Update the image, title, description, and date.

🎥 Swap the YouTube video

Open watch.html. Find the <iframe>. Replace the video ID in the URL with yours.

📱 Update social handles

Search-and-replace @ottogusshow across all HTML files. Update the href on each social link.
